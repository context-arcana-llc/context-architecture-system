# Context Change Log

A running log of significant changes to this context system.

Git commit history covers the file-level details. This file covers the *why* — the reasoning, the trigger, the intent behind each meaningful change.

---

| Date | What changed | Why |
|---|---|---|

---

## What counts as significant

Add a row when:
- A folder's purpose or scope changes
- A key file is added, heavily revised, or archived
- A decision is made that affects how context is structured
- A model or agent gave bad output because the context was missing or wrong — and you fixed it

Skip it for typo fixes, formatting tweaks, or minor additions.
