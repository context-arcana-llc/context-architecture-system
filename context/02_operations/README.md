# 02 — Operations

How the business or practice runs day-to-day.

**What belongs here:**
- Tool and software inventory (see `templates/account-inventory.md`)
- Account ownership records (which accounts exist, who owns them — no credentials)
- Vendor and subscription list
- Meeting cadences and communication norms
- Organizational structure and role definitions

**What does not belong here:**
- Passwords, API keys, or credentials (use a password manager)
- Step-by-step process instructions (→ `06_workflows/`)
