# 09 — Outputs

AI-generated or AI-assisted outputs worth keeping.

**What belongs here:**
- Drafts and finished pieces produced with AI assistance that represent your voice and are worth referencing
- Outputs that demonstrate what "good" looks like for a given task
- Automated agent logs (if you run agentic workflows against this repo)

**What does not belong here:**
- Everything. Be selective — only outputs that have future reference value.
- Raw, unreviewed AI output without any human editing or judgment applied

**Promotion path:** When an output graduates to a reusable template or approved asset, move it to `05_assets/`.
