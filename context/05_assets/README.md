# 05 — Assets

Reusable content and finished material.

**What belongs here:**
- Approved copy blocks (bios, taglines, boilerplate)
- Finished templates and documents
- Email sequences and message scripts
- Social media copy
- Reference links and curated resources

**What does not belong here:**
- Large binary files (images, videos) — reference them by path or URL instead
- Drafts and works-in-progress (→ `09_outputs/` until approved)
