# 04 — Clients

Client and relationship records.

**What belongs here:**
- Client profiles (who they are, their context, their goals)
- Engagement history and notes
- Specific agreements or commitments made
- Communication preferences

**What does not belong here:**
- Secrets or login credentials belonging to clients
- PII that doesn't need to be in a Git repo (consider whether this folder should be in a private repo or excluded from remote push)

**Privacy note:** If this repo is or may become public, keep client information in a separate private repo or local-only folder and reference it from here without committing the content.
