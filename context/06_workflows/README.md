# 06 — Workflows

Step-by-step processes and standard operating procedures.

**What belongs here:**
- SOPs for repeatable business processes
- Onboarding sequences (client, vendor, team)
- Checklists for recurring tasks
- Runbooks for tools or systems you operate
- Decision trees for common situations

**Format guidance:**
Use numbered steps. Include the expected outcome of each step. Note any tools or accounts required. Keep each SOP in its own file, named for what it does (e.g., `client-onboarding.md`, `invoice-process.md`).
