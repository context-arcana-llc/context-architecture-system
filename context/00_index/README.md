# 00 — Index

The master table of contents for this context system.

Add a single index file here (e.g., `index.md`) that maps what's in the system, what's current, and what's stale. Update it whenever you add or retire significant context.

**What belongs here:**
- A master index of active context files with one-line descriptions
- A brief summary of the context system's current state
- Pointers to high-priority or frequently referenced files

**What does not belong here:**
- The actual context content (that goes in the numbered folders)
- Secrets or credentials
