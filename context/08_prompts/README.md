# 08 — Prompts

Calibrated, working prompts worth keeping.

**What belongs here:**
- Prompts that produce reliably good output and are worth reusing
- System prompts for specific AI configurations
- Persona definitions for AI assistants
- Prompt variations for different contexts or models

**Use the template:** `templates/prompt-card.md`

**What does not belong here:**
- One-off exploratory prompts
- Prompts that embed secrets or client PII

**Naming convention:** Use descriptive kebab-case names (e.g., `client-proposal-draft.md`, `weekly-review-prompt.md`).
