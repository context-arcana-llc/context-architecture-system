# 99 — Archive

Retired content kept for reference.

Move files here when they are no longer current but may still be useful for historical context. Do not delete — the archive is cheaper than reconstruction.

**What belongs here:**
- Superseded versions of strategies, offers, or workflows
- Context from completed or ended client relationships
- Old prompts replaced by better versions
- Any file you're not ready to delete but no longer actively use

**Convention:** Keep the original filename. Add a date prefix if helpful (e.g., `2025-03_old-pricing.md`).
