# Riverstone Coffee Roasters — Example CAS Shell

This is a worked example of a business CAS shell. Riverstone Coffee Roasters is a fictional small-batch coffee roastery used to demonstrate what a populated context repository looks like.

Browse the files here before setting up your own shell. When you're ready, delete this entire `examples/` folder — it's not part of your context.

---

## What's here

| File / Folder | What it shows |
|---|---|
| `CONTEXT.md` | Shell manifest with YAML frontmatter and model permissions |
| `context/00_index/` | A master index — the map of everything in the shell |
| `context/01_identity/` | Brand voice rules and founding story |
| `context/02_operations/` | Account inventory |
| `context/03_offers/` | Retail subscription and wholesale program |
| `context/06_workflows/` | A step-by-step roast cycle SOP |
| `context/07_decisions/` | Two decision records |
| `context/08_prompts/` | A working prompt card for generating wholesale quotes |
