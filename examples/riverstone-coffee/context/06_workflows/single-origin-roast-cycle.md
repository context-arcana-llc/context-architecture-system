# Single-Origin Roast Cycle

The standard weekly roast workflow. Tuesday mornings. ~6 batches per day, ~30 lb finished per batch on the 5kg roaster (will change after the 12kg upgrade — see `context/07_decisions/2026-03_drum-roaster-upgrade.md`).

---

## Inputs required

- Green coffee, weighed in advance Monday afternoon
- Roast profile from `context/05_assets/roast-profiles/` (one per origin)
- Empty buckets (one per batch)
- Roast log sheet

---

## Workflow

### Pre-roast (30 min before first batch)

1. **6:30 AM** — Turn on roaster, set initial temp to 380°F. Let it stabilize for at least 25 minutes.
2. **6:45 AM** — Pull green coffee from the staging area. Confirm batch weights match the day's plan.
3. **6:55 AM** — Run one warm-up batch with the cheapest available green (usually a Brazil natural). This batch is sold as "Mill Blend" or composted if quality is off.

### Per batch

1. **Charge** — Drop green into the drum at 400°F. Note charge temp on log.
2. **Drying phase** — First 4–5 minutes. Heat at 100%. Watch for color shift from green to yellow.
3. **First crack** — Usually at ~9:00–9:30 minutes. Note exact time and temp.
4. **Development** — Hold for 1:30 to 2:00 after first crack, depending on origin. See profile.
5. **Drop** — Dump into cooling tray. Note drop temp and total time.
6. **Cool** — Stir cooling tray until cool to touch (about 4 minutes).
7. **Log** — Record charge weight, drop weight (loss should be 14–17%), times, and any notes.

### Between batches

- Let drum return to charge temp (~3 minutes)
- Brief next batch profile aloud if origin is changing

### End of day

1. Run one cleanup batch with rice (cleans drum residue)
2. Turn off roaster, leave fan running for 30 minutes
3. Sweep chaff tray, empty into compost
4. File roast logs in the binder

---

## Quality checks

- **Color:** Agtron 55–65 typical for our profiles, measured next morning after rest
- **Cup:** Wednesday afternoon table cupping of each new origin
- **Reject if:** scorched (visible black tips), under-developed (sour/grassy in cup), or moisture loss outside 14–17%

---

## Common issues

- **First crack late (>10 min)** — usually means charge temp too low or batch size too big. Note in log; adjust next batch.
- **Batch tips dark, body light** — drum too hot, airflow too low. Reduce gas, increase fan.
- **Inconsistent loss** — scale needs calibration, or moisture in the green is varying. Check sample bag.
